# Contributing to Conference Timer

Thank you for your interest in improving **Conference Timer**.

This project is intended as a **clean, reliable, and distraction-free** tool for academic and professional events. Contributions are welcome as long as they respect this philosophy.

---

## Ways to Contribute

You can contribute by:

- Reporting bugs or usability issues
- Suggesting improvements or new features
- Submitting pull requests for enhancements
- Improving documentation or accessibility

---

## Design Principles

When contributing, please keep the following in mind:

- The **timer screen must remain distraction-free**
- No advertisements, pop-ups, or auto-redirects
- Must work well on **projectors, tablets, and laptops**
- Prefer **plain HTML, CSS, and JavaScript**
- Avoid heavy frameworks unless clearly justified
- Maintain offline compatibility where possible

---

## Suggested Areas for Improvement

Some ideas for contributors:

- Full-screen toggle button
- Export / import session settings (JSON)
- Multiple sessions per day
- “Lock screen” mode for display-only operation
- Remote control support (phone controlling tablet; backend required)
- Accessibility options (high contrast mode, larger fonts)

---

## How to Submit Changes

1. Fork the repository
2. Create a new branch (`feature/your-feature-name`)
3. Make your changes with clear commits
4. Open a Pull Request with a short explanation

---

## Code Style

- Keep code readable and commented where logic is non-obvious
- Avoid unnecessary abstraction
- Prefer clarity over cleverness

---

Thank you for helping improve Conference Timer.
